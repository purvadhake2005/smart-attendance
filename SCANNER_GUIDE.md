## ‚ö†Ô∏è Troubleshooting

### Camera Not Working?
- Check browser permission for camera
- Make sure camera isn't used by another app
- Try refreshing the page
- Try a different browser

### QR Code Not Scanning?
1. Check lighting - move to brighter area
2. Ensure QR code is not damaged or blurry
3. Try holding device at different angle
4. Move closer or farther (6-12 inches ideal)
5. Use manual entry option instead

### "Already Marked" Error?
- Student already marked attendance for today
- They can only be marked once per day
- To add another record, change system date or wait until tomorrow

### "Student Not Found" Error?
- Student ID doesn't exist in system
- Go to Admin Panel and register student first
- Generate QR code for student
- Then try scanning again

## üí° Tips for Best Results

1. **Good Lighting**
   - Use natural light or bright indoor lighting
   - Avoid shadows on QR code

2. **Steady Hand**
   - Keep device still while scanning
   - Don't move around too much

3. **QR Code Placement**
   - Print QR codes clearly
   - Attach to student ID card
   - Keep codes clean and undamaged

4. **Distance**
   - 6-12 inches is optimal distance
   - Too close or too far won't work well

5. **Multiple Students**
   - Use manual entry for faster bulk marking
   - Keep camera running for continuous scanning

6. **Regular Testing**
   - Test scanning before using in class
   - Make sure all QR codes are legible
