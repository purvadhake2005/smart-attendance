## ğŸ› TROUBLESHOOTING LINKS

| Issue | Solution |
|-------|----------|
| **`http://localhost:8000` won't open** | 1) Run `start_server.bat` first<br>2) Wait 2-3 seconds<br>3) Try again |
| **"Connection refused"** | Server not running - run `start_server.bat` |
| **Can't access from other computer** | 1) Use IP not localhost<br>2) Get IP with `ipconfig`<br>3) Both devices on same WiFi |
| **Port 8000 already in use** | Edit `server.py`<br>Change `PORT = 8000` to `PORT = 8001` |
| **Browser won't auto-open** | Manually type: `http://localhost:8000` in address bar |
| **Can't find IPv4 address** | In `ipconfig` output, find "IPv4 Address..." line |
| **Mobile can't access server URL** | 1) Check same WiFi<br>2) Use correct IP<br>3) Type full URL: `http://192.168.x.x:8000` |
