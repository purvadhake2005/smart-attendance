## ğŸ“ Best Practices

1. **Register Students First**
   - Add all students before taking attendance

2. **Print QR Codes**
   - Attach to student ID cards for easy scanning

3. **Good Lighting**
   - Use natural light or bright room for scanning

4. **Regular Backups**
   - Export data weekly as backup

5. **Clear Old Data**
   - Archive old attendance before clearing

6. **Organize by Class**
   - Use Student ID format: `CLASS-ROLLNO` (e.g., `12A-001`)
